# Eduscraper

Education scraper for some Universities. Scrapes the course catalog and the schedule for all courses.

## Universities

### Supported Universities

-   [x] <a href="https://www.utoronto.ca/" target="_blank">University of Toronto</a>
-   [x] <a href="https://umanitoba.ca/" target="_blank">University of Manitoba</a>

## Features

- Automatic course catalog scraping
- Schedule retrieval for enrolled courses
- User-friendly interface for managing scraped data
- Customizable settings for scraping parameters

## Tech Stack

| Scraper | Server | Client | Database |
| --- | --- | --- | -- |

| Beautifulsoup4 | Django | Tailwind CSS | - |



## Installation and Usage

### Installation

```py
git clone https://github.com/fardeenes7/eduscraper.git
cd eduscraper
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

```

### Creating Database
```py
python manage.py makemigrations
python manage.py migrate
```

### Create Superuser
```
python manage.py createsuperuser
```

### Run the server

```py
python manage.py runserver
```

### Run the scrapers
- Head over to http://127.0.0.1:8000/admin/
- Click on Run scraper for all the universities
