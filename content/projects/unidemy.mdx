## Features
- **Dummy NID API:** Created a national ID database and api to be used for academic purposes.
- **Learning Management System:** Working on an in-house learning management system powered by Next.js app router, Next.js server actions ,Youtube API and many more.
- **Dummy Payment Gateway API:** Working on a payment gateway for academic usage.

## Installation and Usage

1.  Clone the repository to your local machine:

    ```bash
    git clone https://github.com/fardeenes7/unidemy.git
    ```

2.  Prepare local environment variables:

    ```bash
    cp .env.example .env.local
    ```

3.  Install the dependencies:

    ```bash
    npm install
    ```

4.  Start the development server:

    ```bash
    npm run dev
    ```
